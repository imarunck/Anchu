# Anchu
Anchu is a Square, Edgy, Sharp font for the Kannada language. The design of the edge font is inspired by display pixels. This gives it a jagged, rigid look. Edge is perfect for use in digital designs. Establishes a modern, technological, pixel-perfect, and square feel. Only an angle of 90Â° is used in this font. Scripts in this font are Latin, and Kannada.\
